# Clip-Reviewer Logo Concepts

This folder contains starter SVG logo ideas that match the current Clip-Reviewer style:

- Dark base
- Twitch-purple accents (`#9147ff`, `#772ce8`)
- Workflow signals (clip, review, edit, export)

## Concepts

1. `logo-concept-1-frame-check.svg`
- Film/frame + checkmark.
- Best as primary app icon.

2. `logo-concept-2-timeline-cut.svg`
- Vertical frame + timeline split.
- Best for editor-focused branding.

3. `logo-concept-3-monogram-cr-play.svg`
- CR monogram + play glyph.
- Best for social avatar/icon.

4. `logo-concept-4-layout-stack.svg`
- Camera/gameplay/third-area layout motif.
- Best for product page hero branding.

5. `logo-concept-5-wordmark.svg`
- Icon + "Clip-Reviewer" wordmark.
- Best for homepage/navbar branding.
